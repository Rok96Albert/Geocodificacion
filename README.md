# Geocodificacion
Es una Herramienta que permite capturar datos de coordenadas geográficas por medio de la consola y que después se geocodifican para  obtener una dirección textual de las coordenadas obtenidas a traves de un dispositivo móvil. El programa se divide en 2 partes, la primera parte es la captura de datos que se desarrolla en Java y la segunda parte, es la geocodificación que se implementa en Python. 

Requisitos del programa:
Para usar el programa es necesario contar la última actualización de python y el JDK de Java en la versión actual o minimo tener Java SE 17 en adelante.
Instalar anaconda, cuando anaconda es instalado automaticamente se instala el Prompt de anaconda. 
Dentro del prompt, se debe crear un entorno de trabajo y después instalar las dependencias de python con el comando conda. Las dependencias a instalar son:
Shapely.
Geopandas.
Geopy.
Pandas.
Geodatasets.

Algunas dependencias como matplotlib y numpy, ya vienen instaladas con el SDK de python, solo es necesario verificar la versión y actualizar la versión en caso de ser necesario. Dentro del entorno en el prompt de anaconda, se hace la instalación de python. En el caso de Java, se debe consultar la versión que maneja el ordenador y de ser necesario actualizar a la versión más actual. Si el usuario no cuenta con java, deberá instalar java como prerequisito.
